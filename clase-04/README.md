# Clase 04 - Git desarrollo colaborativo

## Stash 

* Existe dentro de git
* No se puede subir al remoto
* Trabaja con una estructura de pila

## Listar los stashes

```sh
git stash list
```

## Como crear stashes

```sh
git stash
```

## Recuperar el stash

```sh
git stash pop # Recupeara el Ãºltimo stash realizado y si no hay conflicto lo borra.
```
